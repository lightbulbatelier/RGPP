Bulletin News   
April 21st, 2021

MANDATORY UPDATE! ALPHA3 RELEASED!

Howdy, everybody! I have some news for you: a new version of BlackBox SDK is already out! This does mean this will be the last Bulletin for Alpha 2 users. Introducing Alpha3, a more polished experience of the Game Guide resurrection. The files can be downloaded from this GitHub mirror.

Just for this time, please delete your HTML folder before applying the patch to RealArcade. Better said, reinstall a fresh copy of RealArcade 1.4, remove the HTML folder and patch your installation with BlackBox SDK Alpha3.

CONFIRMING THE NEW NAME OF THE GAME GUIDE RESURRECTION

It's not a surprise that I was experimenting with a new name for the game guide. After all, BlackBox was just a codename. The chosen name will be Starlight, this will become the service you see first when you run RealArcade with BlackBox SDK Alpha3. For this run, 17 game pages are now available both locally and online. Several pages are still broken due to important changes in the common game page framework. These will be fixed in the meantime.

I sincerely hope you enjoy this release and expect more updates soon!
