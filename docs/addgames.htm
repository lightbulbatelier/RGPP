Add Games
After you've installed a new CD game or retail game demo, click Find Installed Games and Arcade will look for the game on your computer. If Arcade finds the game, it will be added to the My Games section in Arcade.

If Arcade can't find the game, you can manually create a shortcut to the game inside Arcade:

1. Drag-and-drop the icon for starting the game onto any gray-colored part of Arcade.

2. A shortcut to the game will appear in the window below.

3. To play the game from the window, double-click it.
